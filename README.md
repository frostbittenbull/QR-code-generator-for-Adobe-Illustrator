# Генератор QR-кода для Adobe Illustrator

## 📥 Лёгкий и мощный скрипт (JSX) для создания векторных QR-кодов прямо внутри Adobe Illustrator.

Больше не нужно вставлять растровые изображения из сторонних сервисов — генерируйте чистую векторную геометрию QR-кода в один клик прямо в вашем документе.

---

## ✨ Основные возможности

- **Поддержка любого текста:** URL, e-mail, номера телефонов, произвольные строки — кодируется всё что угодно с поддержкой кодировки UTF-8.
- **Автоматический выбор версии:** тип QR-кода (версия 1–40) подбирается автоматически в зависимости от длины и содержимого введённого текста.
- **Выбор уровня коррекции ошибок:** четыре уровня на выбор — L (~7%), M (~15%), Q (~25%), H (~30%) — позволяют балансировать между размером кода и его устойчивостью к повреждениям.
- **Чистая векторная геометрия:** каждый модуль QR-кода — отдельный векторный прямоугольник, пригодный для масштабирования, вырезки и печати любого формата.
- **Умное позиционирование:** код автоматически центрируется по активному артборду.
- **Группировка объектов:** все элементы QR-кода объединяются в именованную группу «QR-код» для удобства работы.

---

## 🛠 Установка

1. Скопируйте файл `QR-code.jsx` в папку скриптов Adobe Illustrator:
   > **Windows:** `C:\Program Files\Adobe\Adobe Illustrator [версия]\Presets\[язык]\Scripts`
   > **macOS:** `/Applications/Adobe Illustrator [версия]/Presets/[язык]/Scripts`
2. Перезапустите Adobe Illustrator.

---

## 📖 Как использовать

1. Откройте документ в Adobe Illustrator и выберите нужный артборд.
2. Перейдите в меню `Файл → Сценарии → QR-code` (`File → Scripts → QR-code`).
3. Введите текст или ссылку в поле ввода.
4. Выберите уровень коррекции ошибок из выпадающего списка.
5. Нажмите **«Сгенерировать»** — QR-код появится по центру артборда.

---

## ⚙️ Уровни коррекции ошибок

| Уровень | Восстановление данных | Когда использовать |
|---|---|---|
| **L** | ~7% | Чистая печать, идеальные условия |
| **M** | ~15% | Стандартное использование |
| **Q** | ~25% | Код может быть частично закрыт (логотип) |
| **H** | ~30% | Суровые условия, повреждённые поверхности |
